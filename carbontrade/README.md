CarbonTrade - Quick Summary
What is CarbonTrade?
CarbonTrade is a retail carbon credit trading platform that allows individuals to buy, sell, and trade Certified Emission Reduction (CER) tokens. Think of it as a "stock market for carbon credits" where each CER represents 1 tonne of CO₂ offset.
Key Features

Trade CER Tokens: Buy/sell carbon credits like Bitcoin
Carbon Calculator: Calculate your personal carbon footprint
Real-time Pricing: Dynamic market prices ($20-$30/CER)
Transaction History: Track all your trades
User Accounts: Secure registration and login

Technology Stack
Backend: Java 21 + Spring Boot 3.2
Architecture: RESTful API
Storage: In-memory 
Build Tool: Maven
